# Wiki Cards

Exercise using [wikipedia's api](https://en.wikipedia.org/w/api.php) to pull a random articles' title, snippet, and link. Displays in a 'wiki card'.

**Uses**:
- JQuery
- Neat
- Yoeman
- Gulp
